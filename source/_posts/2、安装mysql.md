相关设置：  

> 1、添加PATH参数：(同linux)

> > echo "export PATH=$PATH:/usr/local/mysql/bin" >> .bash\_profile
> > 
> >   
> 
> 2、添加root密码

> > sudo mysqladmin -u root password "123456"
> 
> >   
> 
> 3、登陆

> > mysql -h 127.0.0.1 -u root -p
