sudo vim /etc/default/locale

将下面的内容修改

LANG="zh\_CN.UTF-8"

LANGUAGE="zh\_CN:zh"

修改为：

LANG="en\_US.UTF-8"

LANGUAGE="en\_US:en"

  

 sudo vim /var/lib/locales/supported.d/local

改成

en\_US.UTF-8 UTF-8

  

重启  sudo locale-gen

  

\-------------------------------------------------------------------------------------------  

  

若出现如下问题  

        LANGUAGE = (unset),  
        LC\_ALL = (unset),  
        LANG = "zh\_CN.UTF-8"  
  
运行：  
debian:~# apt-get install locales  
debian:~# dpkg-reconfigure locales  
debian:~# locale
